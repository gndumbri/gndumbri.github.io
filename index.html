Intro

My work in the field of Data Science has consisted of Exploratory Data Analysis, and Supervised Machine Learning. I have worked with several Python packages, notably: Pandas, Numpy, Scipy, Sklearn, Matplotlib, Seaborn, to name a few. Machine Learning models I have worked with are Regression: Linear Regression (OLS, Ridge, LASSO, ElasticNet) and Classification/Regression: Logisitc Regression, Support Vector Machines, GradientBoost, RandomForest, and K-Nearest Neighbors. I have used primarliy Python and SQL, with Python being the primary language for the machine learning work I have done. Check out my work.
